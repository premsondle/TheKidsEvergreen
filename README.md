# The Kids Evergreen - Play School Website

A modern, responsive Next.js website for The Kids Evergreen play school.

## Features

- Responsive design for all devices
- Modern UI with Tailwind CSS
- Programs showcase (Play Group, Nursery, L.K.G, U.K.G)
- School features and facilities
- Contact form for inquiries
- Gallery section
- Bilingual content (English & Hindi)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Customization

- Update content in component files under `/components`
- Modify colors in `tailwind.config.ts`
- Add images to `/public` folder and update Gallery component
